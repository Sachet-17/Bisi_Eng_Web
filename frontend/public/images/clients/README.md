## Logo Specifications

- **Format:** PNG with transparency (preferred) or JPG
- **Recommended Size:** 200x100px to 400x200px
- **Aspect Ratio:** Flexible (logos will be scaled to fit)
- **Background:** Transparent or white background preferred
- **File Naming:** Use lowercase with hyphens (e.g., `ntpc.png`, `power-max.png`)

## How to Add Logos

1. Extract individual logos from the source image
2. Save each logo with the filename specified above
3. Place all logo files in this directory: `/public/images/clients/`
4. The website will automatically display the logos on the Clients page

## Logo Extraction Tips

If extracting from a single image file:
1. Use image editing software (Photoshop, GIMP, etc.)
2. Crop each logo individually
3. Save as PNG with transparency if possible
4. Ensure logos are clear and readable at the display size
5. Maintain aspect ratio to avoid distortion
